# FP MCI 2026 - Customer Experience Analysis

Project ini dibuat untuk Final Project Lab MCI 2026 dengan persona Customer Experience Analyst.

## Business Problem

DustiniaDelixia Groceria ingin memahami mengapa review score pelanggan sulit meningkat dan faktor apa yang berhubungan dengan review rendah.

## Main Focus

- Review score analysis
- Delivery delay analysis
- Seller performance
- Product category analysis
- Region analysis
- Customer experience dashboard

## Tech Stack

- Apache Airflow
- ClickHouse
- Metabase
- Python
- Pandas
- Docker
